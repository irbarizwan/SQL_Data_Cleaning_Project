# SQL Data Cleaning Project

## Objective
Clean and preprocess raw dataset by removing duplicates, handling missing values, and normalizing data.

## Tools Used
- SQL (MySQL / PostgreSQL / SQL Server)

## How to Use
1. Open `housing_data cleaning.sql` in your SQL editor.
2. Run the script on your database containing raw data.
3. Script will:
   - Remove duplicate rows
   - Handle missing/null values
   - Normalize inconsistent values

## Notes
- Replace table names in the script according to your database.
- Script is modular and can be reused for other datasets.
